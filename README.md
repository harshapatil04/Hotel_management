#### ** Hotel Management System (SQL Project)
📌 Overview

The Hotel Management System is a database project developed using SQL to manage hotel operations such as customer registration, room booking, payment processing, and employee records.

This project demonstrates database design, relationships using foreign keys, and SQL queries including JOIN, GROUP BY, and aggregate functions.

🗂️ Tables Used

customers – Stores customer details

rooms – Stores room information and availability

bookings – Manages booking records

payments – Tracks payment details

employees – Stores employee information

🔗 Relationships

One customer → Many bookings

One room → Many bookings

Each booking → One payment

Bookings linked using foreign keys

🛠️ Technologies Used

SQL

MySQL / PostgreSQL

RDBMS
